About The Conference

The innovative city of San Francisco, California will be the hosting the Conference!

Discover new technology to expand your knowledge as a developer. Learn about best practices to use when developing web applications from a lineup of industry professional speakers.

The Speakers

Want to meet and network with some of the best and brightest experts, developers, and hackers in the tech industry from around the world?

Our speaker lineup was just annouced, and the response has been phenominial, so better hurry and grab your tickets before we are sold out!

The Conference is going to be one of the most spectacular tech conferences of the decade... so you don't want to miss it!

Topics Include

Node.js: JavaScript runtime React: JavaScript library for user interfaces Angular: web application platform ES2015: ECMAScript 2015 programming Bootstrap 4: web application framework GitHub: version control service using git
